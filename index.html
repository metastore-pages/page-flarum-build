Flarum Build Factory
Пакеты Документация Сообщество

Flarum Standard

Стандартная чистая сборка Flarum, без дополнений. Сборка включает в себя:

  • metastore/flarum-ext-russian - русский языковой пакет.

Flarum Extended

Расширенная сборка Flarum. Сборка включает в себя:

  • metastore/flarum-ext-russian - русский языковой пакет.
  • flagrow/bazaar - система установки расширений в панели администратора.
  • flagrow/byobu - приватные темы.
  • flagrow/upload - система загрузки файлов.

Сборка и распаковка

Сборка происходит каждый понедельник с 02:20 по 02:40.

Сборки представляют собой архивы формата *.tar.xz. Для их распаковки можно воспользоваться программой 7-Zip.

  • Фонд METADATA
  • Репозиторий METASTORE
  • Сообщество UNIX
  • Сообщество вебмастеров
